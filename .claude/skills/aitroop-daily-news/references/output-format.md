# 报告输出格式

纯文本列表格式，简洁直接，多用 emoji。

---

## 标准格式

```
📰 AI Daily · YYYY年M月D日

1. 标题
链接
摘要（50字内，多用emoji）

2. 标题
链接
摘要（50字内，多用emoji）

...
```

**规范：**
- 纯文本，无 Markdown 标题/粗体/分割线
- 每条新闻：标题 → 链接 → 50字内摘要，三行
- 摘要多用 emoji 增强可读性
- 按重要性排序，不分类
- 底部一行：`来源：smol.ai + Karpathy 博客列表 | YYYY-MM-DD`

---

## 格式规范

### 文章卡片

```
### [{标题}]({链接})

{摘要正文，2-4 句}

**关键信息**: {标签1}, {标签2}, {标签3}

---
```

- 标题：中文意译，≤25 字，不加引号
- 摘要：直接陈述事实，不用"本文..."开头，保留具体数据和技术名词
- 关键信息标签：3-5 个，公司名/技术名/主题词混合，中英文均可
- 每个分类的最后一张卡片不需要 `---` 分隔线（直接接下一个分类标题）

### 核心摘要

- 3-5 条，每条以 `- ` 开头
- 每条一句话，不超过 50 字
- 覆盖当日最重要的独立事件，不互相重复

### 分类标题

只输出有内容的分类，无内容的分类直接跳过。

### hashtag 关键词

底部关键词提取今日所有文章中频次最高的主题词，5-10 个。
